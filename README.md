# 📝 Simple Task Manager

A basic Task Manager app to add, delete, and manage your daily tasks using Local Storage.

## ⚙️ Features

- Add new tasks  
- Delete completed tasks  
- Clean and minimal UI  

## 🔧 Tech Stack

- HTML  
- CSS  
- JavaScript
